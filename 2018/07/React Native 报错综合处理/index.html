No bundle URL present

使用 ignite 脚手架构建时候出现此错误，目前可行的解决方法是：
1、删除 iOS/build 目录。
2、同时打开两个iterm 的窗口，一个在项目目录下执行 react-native start，另外一个执行 react-native run-ios即可。
